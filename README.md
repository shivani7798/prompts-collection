## Audio Customization

### Podcast Editing Prompt
```
Act as an audio editing assistant. Help me create a professional podcast by:
1. Suggesting intro and outro music styles
2. Recommending optimal audio levels and equalization settings
3. Providing tips for noise reduction and clarity
4. Advising on segment transitions and pacing
```

### Voice-over Script Generator
```
Create a professional voice-over script for [topic]. 
Include:
- Engaging opening hook
- Clear and concise main content
- Natural pauses and emphasis points
- Call-to-action at the end
Tone: [professional/casual/enthusiastic]
Length: [duration] minutes
```

### Audio Transcription Cleanup
```
Clean up this audio transcription by:
1. Removing filler words (um, uh, like)
2. Correcting grammar and punctuation
3. Organizing into logical paragraphs
4. Maintaining the speaker's natural voice and meaning
```

## Image Generation

### DALL-E / Midjourney Prompt Template
```
Create a detailed image generation prompt for:
Subject: [main subject]
Style: [artistic style, e.g., photorealistic, watercolor, digital art]
Mood: [mood/atmosphere]
Lighting: [lighting conditions]
Composition: [camera angle, framing]
Details: [specific elements to include]
Example: "A serene mountain landscape at golden hour, watercolor style, warm and peaceful mood, soft diffused lighting, wide angle shot, include a small cabin and a winding path"
```

```
- Style references
```

1. **Copy the relevant prompt** for your task
2. **Customize** the bracketed placeholders with your specific information
3. **Paste** into your AI assistant (ChatGPT, Claude, etc.)
4. **Iterate** on the output by asking follow-up questions or requesting modifications
